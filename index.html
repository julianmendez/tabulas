Tabulas

System to manage human-readable tables using files

Tabulas is a system to manage human-readable tables using files. It uses a specific type of file format that is similar to a Java .properties, but allows defining the same property for different objects. Tabulas is an experimental semi-automatic Scala reimplementation of Tabula, which is implemented in Java.

Source code

To clone and compile the project:

$ git clone https://github.com/julianmendez/tabulas.git
$ cd tabulas
$ mvn clean install
